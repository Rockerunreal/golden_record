Инструкция по запуску:
1) расположить файл в папке с data.csv (ваши данные)
2) открыть файл в IDE (VS) и запустить
3) результат будет помещён в RESULT.txt в той же папке что и файл
4)если вдруг вылетает ошибка и он не находит файл, укажите путь к файлу напрямую: 14 строка - data.csv, 16 строка - выход RESULT.txt
5) Алгоритм прописан не до конца, не хватило буквально чуть чуть
что не дописали:
многомерная сортировка(увеличит точность)

Папка наработки:
Есть отдельно работающие валидатор данных, графический интерфейс и алгоритм. Планировалось их объеденить в формате запуска их как отдельных серверов, но не уложились по времени.
